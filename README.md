# House-Price-Prediction-Using-Linear-Regression
This project is a beginner-friendly House Price Predictor built using Python and Linear Regression from the scikit-learn library. It estimates the price of a house based on three input features: area (in square feet), number of bedrooms, and the age of the house. The goal of this project is to demonstrate the basic workflow of a machine learning model, including data preparation, training, and prediction.

To run this project, make sure you have Python installed along with the required libraries: NumPy and scikit-learn. You can install them using pip:

pip install numpy scikit-learn

After installing the dependencies, simply run the Python script in your terminal or code editor. The program will prompt you to enter the area, number of bedrooms, and age of the house. Once you provide the inputs, the model will predict and display the estimated house price in lakhs.

This project uses a small, manually created dataset for simplicity, so the predictions are approximate and meant for learning purposes only. It is a great starting point for beginners exploring machine learning and can be further improved by using real-world data and adding more features.
